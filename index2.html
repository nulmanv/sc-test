Security Policy

Privacy Preservation

  • Use anonymization techniques when storing user data
  • Limit data retention to only what is legally required
  • Provide users with access and deletion options for their data
  • Avoid logging sensitive information unless explicitly needed
  • Apply strict access control to privacy-related logs

General Security Guidelines

  • Keep server software and libraries up to date
  • Use multi-factor authentication for administrative access
  • Secure physical access to data centers and workspaces
  • Regularly train employees on security awareness
